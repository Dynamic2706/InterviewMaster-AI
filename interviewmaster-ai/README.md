# InterviewMaster AI
AI-Powered Mock Interview Simulator with Evaluation, Scoring, and Reports.

## Quickstart
1. Install requirements: `pip install -r requirements.txt`
2. Run API: `uvicorn app.main:app --reload`
3. Open `http://127.0.0.1:8000/docs` for interactive API docs.
